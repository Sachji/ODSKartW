Project Task

Students will design and implement a web application in the Java EE environment, which will allow the management of media with video content. A sheet in ODF format will be used as the source and will be provided by the project supervisor.

- adding a medium to a category

- adding a new category

- searching for a movie

- removing a medium from a category

- removing a category

- changing the category of a medium

Applications will be implemented as a web application with the use of JSP / Servlet container Jetty.

Main window

The main window of the application interface offering the exploration of the attached ODF file or the search of a specific movie.

Figure 3.1. Main window

Buttons:

Explore - explores the attached ODF file

Search for movie - initiates a search query for a specific movie


Search

Allows searching for a specific movie by typing its name into a text box.

Figure 3.2. Search window

Buttons:

Submit - confirms the specified search query


Search results

A window that lists the results of the specified search query.

Figure 3.3. Search results window

Buttons:

Move - moves the specified medium to another category

Delete - deletes the specified medium


Categories list

The part of the interface that is used to manage categories. Lists all the categories with their ID number, name and the maximum number of media in them.

Figure 3.4. Categories

Buttons:

Add Category - adds a new category

Delete - deletes the specified category


Media list

The part of the interface that is used to manage media. Lists all the media of a category with their ID and content.

Figure 3.5. Media

Buttons:

Add Medium - redirects to a window that allows the addition of a new medium to the category

Move - moves the specified medium to another category

Delete - deletes the specified medium


Adding a medium

Manages the addition of media to a category. The names of the media to be added are specified here.

Figure 3.6. Add medium

Buttons:

Submit - confirms the addition of a new medium to the category
